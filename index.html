Object Relational Mappers

Sequelize

Sequelize is a promise-based Node.js ORM that can be easily integrated with PostgreSQL that's loaded with features like solid transaction support and relational mapping. Because of this Sequelize is my go to when querying data from my PostgreSQL database.

SQLAlchemy

But what about when we build our backend in Python? Easy Flask allows us to hook up SQLAlchemy in Python and connect it to PostgreSQL to easily create tables and relationships and quickly and efficently query for data.
